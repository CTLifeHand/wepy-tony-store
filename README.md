# wepy-tony-store
wepy store tony

### 项目构建
npm install wepy-cli -g

- 同样有bug 一旦有wpy 就会被编译 等于有错误的代码会自动失败

### 开发使用说明

使用微信开发者工具新建项目，本地开发选择dist目录。

微信开发者工具 --> 项目 --> 关闭ES6转ES5。

本地项目根目录运行npm run dev，开启实时编译。
